# Kişisel Bilgi Sistemi

## Açıklama
Kullanıcıdan ad, soyad, yaş, şehir ve meslek bilgilerini alıp düzenli bir formatta gösteren Python programı.

## Öğrenilen Konular
- Değişkenler
- `input()` fonksiyonu
- `print()` ve f-string formatlaması
- String çarpma işlemi

## Nasıl Çalıştırılır
```bash
python kisisel_bilgi.py
```

## Örnek Çıktı
```
========================================
KİŞİSEL BİLGİLER
========================================
Adı-soyadı  : Ahmet Yılmaz
Yaş         : 25
Şehir       : İzmir
Meslek      : Mühendis
========================================
```
