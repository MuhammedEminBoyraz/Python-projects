# Öğrenci Not Sistemi

## Açıklama
Öğrenci ekleme, listeleme ve en başarılıyı bulma sistemi.

## Özellikler
- Öğrenci ekleme (ad + 3 sınav notu)
- Otomatik ortalama hesaplama
- Harf notu atama (AA, AB, BB, CB, FF)
- Tüm öğrencileri listeleme
- En başarılı öğrenciyi bulma
- Menu sistemi

## Kullanım
```bash
python ogrenci_not.py
```
